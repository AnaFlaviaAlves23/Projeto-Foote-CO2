# Projeto Foote: Calculadora de CO2

Avaliação experimental do impacto do CO2 na temperatura atmosférica, baseada no modelo de Eunice Foote — desenvolvido para a **10ª FCT EEICP** (Feira de Ciência e Tecnologia).

🔗 **Site publicado:** [projeto-foote-co2.netlify.app](https://projeto-foote-co2.netlify.app)

## Sobre o projeto

Este site apresenta uma calculadora interativa de emissão de CO2 por trajeto, além de conteúdo educativo sobre o efeito estufa, a descoberta histórica de Eunice Newton Foote (1856) e o ciclo do carbono no planeta.
